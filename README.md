# Sales Performance & Profitability Analysis

## Project Overview
This project analyzes sales performance and profitability for a multi-country retail company using Power BI.  
The goal is to identify key trends, performance drivers, and actionable business insights to support data-driven decision-making.

## Business Questions
- How are sales and profit evolving over time?
- Which countries and regions are the most profitable?
- Which product categories and sub-categories drive revenue and profit?
- How do discounts impact profitability?
- Where are the main growth opportunities and risks?

## Dataset
The analysis is based on a global retail sales dataset including:
- Order and shipment dates  
- Country and region information  
- Customer segments  
- Product categories and sub-categories  
- Sales, quantity, discounts, and profit  

## Tools & Technologies
- Power BI (data modeling, DAX, dashboards)
- Excel / CSV
- GitHub (version control and portfolio presentation)

## Key KPIs
- Total Sales  
- Total Profit  
- Profit Margin (%)  
- Number of Orders  
- Average Order Value  
- Sales Growth (MoM / YoY)  

## Key Insights (to be developed)
- Sales and profitability trends over time
- Performance comparison by country and category
- Identification of high-discount / low-profit products
- Data-driven recommendations for business improvement

## Deliverables
- Interactive Power BI dashboard
- Business-focused insights and recommendations
- Clean and well-documented repository
